Abstract

This project proposes a vector quantized segmentation model for analyzing sequential trajectory data using hierarchical latent representations. By leveraging Residual Vector Quantization with a multi-stage architecture, the model captures both coarse and fine-grained behavior segments. A probabilistic reconstruction loss combined with smoothness penalties ensures interpretable and temporally consistent segment labels. We evaluate the approach on the AntMaze and PointMaze environments, where the segmentation codes reveal meaningful structure aligned with task dynamics.
